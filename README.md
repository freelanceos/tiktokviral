# دليل ربط نموذج "اتصل بنا" بجوجل شيت

هذا الدليل يشرح كيفية ربط نموذج الاتصال بجوجل شيت لتخزين البيانات المرسلة من الزوار.

## الخطوات المطلوبة

### 1. إنشاء جدول بيانات جديد في Google Sheets

1. انتقل إلى [Google Sheets](https://sheets.google.com) وقم بتسجيل الدخول إلى حسابك.
2. أنشئ جدول بيانات جديد.
3. في الصف الأول، أضف العناوين التالية:
   - الاسم
   - البريد الإلكتروني
   - رقم الهاتف
   - الرسالة
   - وقت الإرسال

### 2. إنشاء Google Apps Script

1. في جدول البيانات، انتقل إلى **Extensions > Apps Script**.
2. سيفتح محرر Apps Script في نافذة جديدة.
3. احذف أي كود موجود في الملف الافتراضي.
4. انسخ الكود من ملف `google_apps_script.js` والصقه في المحرر.
5. استبدل `SPREADSHEET_ID` بمعرف جدول البيانات الخاص بك (يمكن العثور عليه في URL جدول البيانات بين `/d/` و `/edit`).
6. احفظ المشروع بتسمية مناسبة مثل "معالج نموذج الاتصال".

### 3. نشر التطبيق كتطبيق ويب

1. انقر على **Deploy > New deployment**.
2. اختر **Type > Web app**.
3. قم بتعيين:
   - **Execute as**: Me
   - **Who has access**: Anyone
4. انقر على **Deploy**.
5. انسخ عنوان URL الذي تم إنشاؤه.

### 4. تحديث ملف JavaScript

1. افتح ملف `script.js` في مجلد `contact_us`.
2. ابحث عن المتغير `scriptURL`.
3. استبدل القيمة الحالية `'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec'` بعنوان URL الذي نسخته في الخطوة السابقة.
4. احفظ الملف.

## اختبار النموذج

1. افتح صفحة "اتصل بنا" في المتصفح.
2. املأ النموذج ببيانات اختبارية.
3. انقر على زر "إرسال الرسالة".
4. تحقق من ظهور رسالة النجاح.
5. افتح جدول البيانات للتأكد من استلام البيانات.

## ملاحظات هامة

- تأكد من أن لديك صلاحيات كافية للوصول إلى جدول البيانات وتعديله.
- قد تحتاج إلى الموافقة على الأذونات عند أول تشغيل للتطبيق.
- إذا واجهت أي مشاكل في CORS، يمكنك استخدام مكتبة [Google Apps Script CORS](https://github.com/tanaikech/taking-advantage-of-Web-Apps-with-google-apps-script) لحلها.

## تخصيص إضافي

- يمكنك تعديل تصميم نموذج الاتصال من خلال ملف `style.css`.
- يمكنك إضافة حقول إضافية للنموذج عن طريق تعديل ملف `index.html` وتحديث كود Google Apps Script وفقًا لذلك.
